## Como adicionar quando cliente enviar:
1. Converter PNG/SVG para WebP usando: https://cloudconvert.com/png-to-webp
2. Salvar em: /public/assets/images/
3. Substituir em todos os arquivos HTML:
   - Header navigation
   - Footer
   - Meta tags (Open Graph)
4. Gerar favicon: https://realfavicongenerator.net/
